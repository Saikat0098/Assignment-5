What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface that represents a web page's structure and allows programs to change its content, style, and structure

How do you select an element from the DOM?

To select an element from the Document Object Model (DOM) in JavaScript, you use methods like getElementById() to access an element by its unique ID, getElementsByClassName() to find elements with a specific class name, or querySelector() to select an element based on a CSS selector
